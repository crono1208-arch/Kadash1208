# Sistema de Cotización de Refacciones

Aplicación desarrollada en **Python** para automatizar procesos de cotización de refacciones, utilizando **interfaz gráfica con Tkinter** y **base de datos SQLite** para optimizar tiempos y organización de información.

## Tecnologías utilizadas

- Python
- SQLite
- Tkinter
- Programación Orientada a Objetos (OOP)

## Funcionalidades

- Generación de cotizaciones de refacciones
- Cálculo automático de costos según cantidad de piezas
- Consulta, almacenamiento y actualización de información mediante SQLite
- Lectura y procesamiento de datos
- Interfaz gráfica para facilitar el uso del sistema
- Automatización de procesos de cotización

## Objetivo del proyecto

Este proyecto fue desarrollado como práctica aplicada de programación en Python, enfocado en resolver una necesidad real relacionada con la automatización de cotizaciones y organización de información.

## Aprendizajes obtenidos

Durante el desarrollo del proyecto trabajé en:

- Programación Orientada a Objetos (OOP)
- Manejo de bases de datos con SQLite
- Desarrollo de interfaces gráficas con Tkinter
- Debugging y resolución de errores
- Organización y lógica de negocio
- Manejo de datos y automatización de procesos

## Próximas mejoras

- Exportación de cotizaciones a PDF
- Mejoras visuales de la interfaz
- Optimización del manejo de datos
- Nuevas funcionalidades para automatización
